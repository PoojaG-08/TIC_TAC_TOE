# 🎮 Tic Tac Toe Game

A modern and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**.  
This project provides a clean user interface, smooth gameplay experience, and responsive design for all devices.

---

## 🚀 Features

- Interactive two-player gameplay
- Responsive design for desktop and mobile
- Winner detection logic
- Draw match detection
- Restart / Reset game functionality
- Simple and user-friendly interface
- Lightweight and fast performance

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & Layout
- **JavaScript (ES6)** – Game Logic

## 🎯 How to Play

1. Player **X** starts the game
2. Players take turns selecting empty cells
3. The first player to align 3 symbols horizontally, vertically, or diagonally wins
4. If all cells are filled without a winner, the game ends in a draw
